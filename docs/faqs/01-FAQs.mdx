# FAQs

## Helixbox Basics

<details>
<summary>What makes Helixbox different from other wallets?</summary>

Helixbox offers a unique combination of features:
- Smart wallet with social login integration
- Multi-chain support through a single interface
- Built-in DeFi tools with advanced analytics
- Enterprise-grade security protocols
- Intuitive experience for both novice and expert users
</details>

## Wallet & Security

<details>
<summary>How secure is my Helixbox wallet?</summary>

Helixbox implements multiple security layers:
- Powered by Privy's secure infrastructure
- Social recovery options available
- Private keys are never stored on our servers
- Multi-factor authentication support
- Regular third-party security audits

:::tip Security Recommendation
Enable all security features in your account settings for maximum protection.
:::
</details>

<details>
<summary>How to keep my assets safe?</summary>

Follow these security best practices:
1. Use strong passwords for social login
2. Enable two-factor authentication
3. Never share login credentials
4. Verify transaction details in the Helixbox interface
5. Securely store recovery information
6. Monitor account activity regularly
</details>

## Transactions

<details>
<summary>How do gas fees work on Helixbox?</summary>

Helixbox offers flexible gas handling:
- Pay gas fees in multiple tokens
- Gasless transactions for supported operations
- Clear gas estimates before confirmation
- Transaction batching to save on fees
</details>

<details>
<summary>Why did my transaction fail?</summary>

Common reasons in Helixbox:
1. Insufficient balance for gas fees
2. Network congestion
3. Slippage tolerance too low
4. Smart contract interaction issues

:::tip
Check the transaction details in your Helixbox dashboard for specific error messages and solutions.
:::
</details>
